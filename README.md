# yaml-mindmap-parser
YAML driven mindmap parser


Mindmap sample in YAML format

```yaml
My Project:
  - []:
    - name: task #1
      description: some text
      url: http://api.skitsanos.com
      []:
      - zz
      - aa
      - name: subtask
        []:
        - tsk1
        - tsk2
    - name: another task
      []:
      - 11
      - 22
      - 33

```
